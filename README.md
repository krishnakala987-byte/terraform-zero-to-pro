# terraform-zero-to-pro

A Terraform lab worked through one module at a time, going from first config up to the pieces you need for real infrastructure. Each numbered folder is a focused step.

## What it covers

Starting from the basics (01-basics) and building up through variables, outputs, and local values, then into resource dependencies, state management, reusable modules, and workspaces. Everything runs against AWS.

The idea is to build the concepts up in order rather than jump straight to a finished project, so each step stays small and readable.
